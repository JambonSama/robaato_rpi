# Robaato RPi code

The ROS packages developped to make our Robaato robot move. This robot was developped in the context of the [EPFL robot competition](https://robot-competition.epfl.ch/ "EPFL robot competition") for the Fall 2020 edition.

The final code run for the competition is available here, developped for ROS Melodic.

Launch the packages in the following order with their launch files:

1. robaato_description
1. robaato_navigation
1. robaato
